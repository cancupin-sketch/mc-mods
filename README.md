## v1.1.0 (2026-03-08)
- 런처 UI 전면 개편 (1280x720, 웹사이트 컨셉 통일)
      . 런처의 UI를 공식 웹사이트와 동일하게 적용했습니다.
  
- 기본 배경 이미지 번들 포함
    . 런처의 기본 이미지가 포함되지 않은 사항을 수정했습니다.
  
- 서버 접속하기 버튼 비활성화 버그 수정
    . 서버 접속하기 버튼이 마인크래프트가 꺼진 이후에 다시 돌아오지 않는 버그를 수정했습니다.
  
- Java 자동 설치 실패 시 안내 메시지 표시
    . 어떻게 해도 자바가 자동 설치되지 않는다면. 웹페이지 수동 설치로 안내합니다.
  
- Java 설치 관리자 권한 요청 추가
    . 기존 런처에 자바 자동 설치가 작동되지 않는 문제의 해결안입니다.
  
- BlueMap 온라인 맵 추가
    . 이제 집 밖에서도 서버를 관음할 수 있습니다.
  
- KubeJS + Rhino 추가
    . 스크립트와 내부 명령어 작성을 위한 툴입니다.
  
- Lightman's Currency 추가
   . 경제를 위한 돈과 상점, 은행을 추가해줍니다.
  
- 튜토리얼 차원 및 스크립트 추가
   . 이제 기본 보급품을 받고 규칙을 이해할 수 있습니다.
  
- FTB Chunks, Teams, Library 추가
   .  모드 단위에서 진행되는 구역 보호입니다.
